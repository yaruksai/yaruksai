# Contributing to YARUKSAİ

Thank you for your interest in contributing! YARUKSAİ is an open-source AI ethics engine, and we welcome contributions from the community.

## How to Contribute

### 1. Report Issues
- Use GitHub Issues for bugs and feature requests
- Include reproduction steps for bugs
- Be as specific as possible

### 2. Submit Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally with `docker compose up -d`
5. Submit a PR with a clear description

### 3. Code Style
- **Python:** Follow PEP 8
- **TypeScript:** Use existing project conventions
- **Commits:** Use clear, descriptive commit messages

### 4. Areas We Need Help
- 🌍 **Translations** — Add new languages to the UI
- 🧪 **Testing** — Unit tests for pipeline stages
- 📝 **Documentation** — Improve API docs and guides
- 🔌 **Integrations** — Connect to more LLM providers
- 🏛️ **Compliance** — Add regulations beyond EU AI Act

## Code of Conduct

Be respectful, constructive, and inclusive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
